◆ transform()

def task_parser.transform (   x,
  y 
)

This transform calculates the two motor angles on the board given some point within the drawing grid.

The point is calculated with respect to the corner closest to motor 1. This assumes the distance between the motors is _R = 263mm and the "zeroed" length of each motor is r = 330 mm (fully extended to reach the furthest corners) which results in a homed location of (_x_home, _y_home) = (182.3, 122.7) mm.

Parameters
xthe x coordinate (mm)
ythe y coordinate (mm)
Returns
the motor angles (r_1, r_2) in (ticks)

◆ DPMM

int task_parser.DPMM = 40

Dots per mm from the Inkscape HPGL output.

By default, Inkscape outputs HPGL files with units of 1/40th of a millimeter, since this is the resolution of the HP plotters which HPGL was originally designed for.

◆ PPR

int task_parser.PPR = 256*4*16

Encoder pulses (ticks) per revolution.

The encoder pulses or ticks per revolution of the pulley is 256 counts per revolution times 4 pulses per count for a quadrature encoder time 16 for the 16:1 ratio between the backshaft of the motor where the encoder is and the output shaft of the gearbox where the pulley is.
